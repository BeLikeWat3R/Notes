
Each line of assembly will have 3 main parts of the code. 

Labels: 

	i.e. message:, etc. 

Instructions: 

	mov, syscall, etc. 

Operands: 

	.data, rax, 1, etc. 

-----------------------------------------

Variables: 

Defined by using db (define bytes), dw (define words), or dd (define digits)

Comment them out so you know what they do
